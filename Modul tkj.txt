■	Masuk ke dalam Mikrotik lalu buat firewall untuk blok VLAN 20 (Siswa) akses ke VLAN 30 (Server). IP → Firewall → Filter Rules → General ( Chain: forward, Src. Address: network siswa VLAN 20, Dst. Address : network server VLAN 30) → Action : drop
 

■	buat firewall untuk mencatat Logging aktivitas penting (logging router VLAN 10,20,30). IP → Firewall → Filter Rules → General ( Chain: forward → Action : log dan ceklis Log
Buat sebanyak 3 untuk Out Interface berbeda yaitu vlan 10, vlan 20, vlan 30

 

 



■	Amankan akses router dari SSH dan Winbox, Akses winbox dan SSH hanya dapat diakses oleh VLAN 10 (Guru_Admin). Ijinkan akses SSH ke VLAN 10 (Guru_Admin) dengan cara: IP → Firewall → Filter Rules → General (Chain: input, Src. Address: 192.168.10.0/24, Dst.
Address: 192.168.200.0/24, Protocol: 6(tcp), Dst. Port: 22) → Action:
accept
 
 



Blok Winbox dari akses VLAN 20 (Siswa) dengan cara IP → Firewall → Filter Rules → General ( Chain: input, Src Address: 192.168.20.0/24, Protocol: 6(tcp), Dst. Port: 8291) → Action: drop


 
■	buat firewall untuk mengizinkan monitoring router ke Cacti.
IP → Firewall → Filter Rules → General (Chain: input, Protocol: udp, Dst. Port: 161) → Action: accept



Catatan:
Seluruh Firewall di Filter Rules dikerjakan setelah konfigurasi server agar tidak mengalami permasalahan koneksi jaringan pada server atau bisa di non aktifkan terlebih dahulu untuk memulai konfigurasi server.
